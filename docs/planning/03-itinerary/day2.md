# Day 2｜09/03（四）：甘川文化村・松島纜車

← [返回每日行程總覽](./index.md)

> **主題：** 西面出發往西區，上午甘川洞拍網美照，下午松島搭海上纜車、走天空步道。
> **服裝建議：** 甘川洞坡道、階梯多，穿好走的鞋；想拍韓服 / 手翻書可帶一套淺色美美服裝替換。

## 注意事項

- 甘川文化村韓服租借、手翻書拍攝約需 10:00 前到場排隊，人多時排隊時間可達 25-30 分鐘。
- 松島海上纜車假日人潮多，建議避開整點尖峰時段，或提早現場購票。
- 松島纜車站離最近公車站尚有一段路程，建議直接搭計程車往返，保留體力。
- 人數 10+ 人，今天西面→甘川洞→松島→南浦/西面 需多段接駁、都不在地鐵沿線，建議考慮包一台半日／全日車，比分搭多台計程車更省事。
