# Day 4｜09/05（六）：海雲台・青沙浦・廣安里

## 注意事項

- 膠囊列車兩條路線「尾浦→青沙浦」較熱門，若搶不到可改訂「青沙浦→尾浦」路線。
- X the Sky、海岸列車是否可用釜山PASS兌換，出發前 1 個月再用官方 APP 確認方案、分組與合作景點。
- 午餐、晚餐都不統一訂位，建議拆成 3-4 組分別在海雲台、廣安里解決，約定時間在集合點會合即可。
