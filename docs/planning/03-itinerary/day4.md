# Day 4｜09/05（六）：機張一日遊（Skyline Luge・樂天世界・Outlet）

← [返回每日行程總覽](./index.md)

> **主題：** 機張郡同樣離西面較遠，安排一整天玩刺激設施＋購物血拼，晚上回西面過最後一夜。
> **服裝建議：** 輕便、運動、好走動（Skyline Luge、樂天世界會較耗體力）

## 注意事項

- Skyline Luge、樂天世界門票建議先線上購買，且皆可持釜山PASS免費入場，記得規劃當日 PASS 使用順序。
- 這是最後一個整天行程，若有想買但還沒買齊的伴手禮，優先在 Outlet／超市這站解決，隔天時間較趕。
- Old Mansion 老宅邸建議先電話或現場確認候位狀況，假日晚餐時段人潮較多。
- 機張郡其實有東海線 Osiria 站可到（三個景點步行 10-20 分鐘內），但西面搭大眾運輸過去需轉乘、抓 1-1.5 小時，比計程車直達（40-50分鐘）慢不少；人數 10+ 人，建議考慮包一台全日車（含西面來回），省時間也省轉乘的麻煩。
