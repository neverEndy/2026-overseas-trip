## 注意事項

- 出發前 3 天內先上網填寫 **e-Arrival Card**。K-ETA 目前有臨時豁免政策，但出發前仍要用官方 K-ETA／航空公司規定確認；Q-CODE 只有特定檢疫情況才需要，不要當成每個人都必填。
- 行李箱較大件建議直接請飯店櫃檯寄放，不用整組拖著逛街。
- 若想申辦 **釜山PASS（Visit Busan Pass）** 實體卡，可留意鄰近據點兌換方式，後面幾天會用到（Skyline Luge、Lotte World、X the Sky、松島纜車、海雲台海岸列車等）。
- 16 人用餐原則：Day 1、Day 3 晚餐嘗試訂位（訂不到就改分流）；其他午餐與晚餐都以指定區域分流，各自找餐廳／小吃，約定時間集合。
