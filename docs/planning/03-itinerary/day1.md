# Day 1｜09/02（三）：抵達釜山・西面＆南浦初體驗

← [返回每日行程總覽](./index.md)

> **主題：** 抵達日，行李寄放後在飯店周邊（西面）與鄰近的南浦洞熱身，不安排太滿，調整時差體力。
> **住宿：** 阿爾班酒店 Arban Hotel（西面站附近）— 詳見 [航班&飯店](../02-booking-and-transfers/index.md)

## 行程總覽

| 時間 | 行程 | 地點 | 備註 |
|------|------|------|------|
| 04:00 | 前往桃園機場 | 桃園機場 第二航廈 | 09/02（三）06:15 CI190 起飛，需提早 2.5 小時到機場 |
| 05:35 | **登機時間** | 桃園機場 CI190 登機門 | 通常起飛前 40 分鐘開始登機，登機門於起飛前 20 分鐘關閉，請提前在登機門候機 |
| 06:15 | 飛機起飛 TPE→PUS | CI190 中華航空 | 飛行約 2.5 小時 |
| 09:40 | **抵達時間**：抵達金海國際機場 | 金海機場 第一航廈 | 韓國時間比台灣快 1 小時，實際落地時間以航班動態為準 |
| 09:40-10:40 | 入境通關、提領行李 | 金海機場 | 建議出發前 3 天先上網完成 Q-CODE 申報，過海關掃 QR code 更快 |
| 10:40-11:30 | 前往飯店寄放行李 | 金海機場 → 西面 | 輕軌＋地鐵：沙上站轉乘，約 50 分鐘；或計程車約 35 分鐘（約 ₩17,000-20,000） |
| 12:00 | 午餐 | 西面周邊 | 西面美食選擇多，先簡單吃，晚上再吃好料 |
| 13:30-15:00 | 西面地下街、樂天百貨/超市 逛逛 | 西面 | 換錢、買生活用品、熟悉飯店周邊環境 |
| 15:00 | 飯店 Check-in | 阿爾班酒店 Arban Hotel | 標準入住時間 15:00，正式放行李、稍作休息 |
| 16:00-18:30 | 南浦洞熱身：BIFF廣場、國際市場、光復路時尚街 | 南浦洞 | 西面搭地鐵 1 號線至南浦站，約 15-20 分鐘 |
| 19:00 | 晚餐 | 南浦洞 或 西面 | 可挑南浦雪濃湯、或西面美食街 |
| 21:00 | 返回飯店休息 | 阿爾班酒店 | 第一天調時差，早點休息迎接明天甘川洞 |

## 注意事項

- 入境前 3 天內先上網填寫 K-ETA／Q-CODE 檢疫申報，海關掃 QR code 通關更快。
- 行李箱較大件建議直接請飯店櫃檯寄放，不用整組拖著逛街。
- 若想申辦 **釜山PASS（Visit Busan Pass）** 實體卡，可留意鄰近據點兌換方式，後面幾天會用到（Skyline Luge、Lotte World、X the Sky、松島纜車、海雲台海岸列車等）。
