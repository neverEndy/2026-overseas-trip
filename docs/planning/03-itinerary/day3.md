# Day 3｜09/04（五）：海雲台一日遊

← [返回每日行程總覽](./index.md)

> **主題：** 距離西面較遠、需專程跑一趟的海雲台區域，安排整天在此，晚上再回西面過夜，避免來回奔波。
> **⚠️ 提前預約：** 天空膠囊列車官網開放預約時間為出發前 4 週的韓國時間週二 09:00（台灣時間週二 08:00），週末場次搶手，務必提早訂！

## 注意事項

- 膠囊列車兩條路線「尾浦→青沙浦」較熱門，若搶不到可改訂「青沙浦→尾浦」路線。
- X the Sky、海岸列車皆可用釜山PASS兌換，記得先確認手上是 24/48 小時方案還是 BIG3/BIG5 限量方案，兌換規則不同。
- 午餐、晚餐都是二選一的湯飯／炸豬排店，不用兩間都排，體力許可再考慮宵夜炸雞。
