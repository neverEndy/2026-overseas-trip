# Day 5｜09/06（日）：南浦最後採買・賦歸

← [返回每日行程總覽](./index.md)

> **主題：** 回程班機 20:00 起飛，時間較晚，白天還能再逛半天，行李可先寄放飯店櫃檯，不用一早就拖著跑。
> **服裝建議：** 輕便舒服為主，方便通關與久坐飛機

## 注意事項

- 若有退稅商品，記得在機場先辦理退稅（Tax Refund）再去報到櫃檯託運行李，部分商品需海關查驗。
- 假日午後往機場的路上／輕軌轉乘可能人潮較多，建議比預估時間再抓寬鬆一些。
- 出發前務必再次確認護照、機票、退稅單據都在隨身行李中。
