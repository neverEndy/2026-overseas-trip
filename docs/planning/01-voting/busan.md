## 票價速覽與結論

以下數字來自 Google Flights 已記錄之「最佳」直達來回起價（1 人、經濟艙、含稅費），**僅供趨勢參考**，訂票前請再查一次。

| 指標 | 內容 |
|------|------|
| **26 檔區間** | 來回約 **$7,452～$15,841**（最低與最高皆在表內） |
| **最便宜** | **09/16（三）–09/20（日）** 起 **$7,452**（台灣虎航直達） |
| **連假尖峰** | **中秋** **$15,841**；**光復** **$12,295**；**聖誕** **$13,637**；**雙十** **$11,109** |
| **7–8 月** | 多數 **$8,900～$10,600**，暑假偏高 |
| **9 月中下旬** | **$7,452～$8,109**，全年最甜區間之一 |
| **10/28 起至 12/20** | 「最佳」常為 **大韓航空** 下午班，約 **$9,041～$10,307** |
| **主力航司** | 台灣虎航、易斯達航空；秋冬另見大韓、聖誕檔有長榮 |

**一句話**：要**直達 + 全檔期都能飛**，釜山最穩；想省錢可鎖 **9/9～9/20** 附近；**連假**三地都會貴，釜山中秋仍明顯跳價。

票價與連結資料來自同資料夾下 `data/busan.json`，列表與圖表由該檔渲染。
